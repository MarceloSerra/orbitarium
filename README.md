# Orbitarium

3D interactive solar system explorer built with React Three Fiber.

## Features

- Realistic 3D rendering of the Sun and 8 planets
- NASA textures with procedural fallback
- Visible orbital paths
- Space background with stars and distant galaxies
- Click any celestial body for detailed information
- Smooth orbital animations with pause/resume controls

## Tech Stack

| Technology | Purpose |
|------------|---------|
| Vite + TypeScript | Build tool and type safety |
| React 18+ | UI framework |
| React Three Fiber | Declarative 3D rendering |
| @react-three/drei | Helpers: OrbitControls, Stars, Text, Html |
| @react-spring/three | Smooth orbital animations |
| Zustand | Minimal global state |
| Framer Motion | Drawer UI animations |
